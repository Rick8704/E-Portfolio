## 📖 What I Did

As part of Group 11, I contributed to the analysis of a selected interface using HCI evaluation methods. The group identified usability strengths and weaknesses, applied heuristic evaluation criteria, and proposed concrete design improvements supported by reasoning.

---

## 💭 Reflection

**What I gained:** Practical experience applying usability evaluation frameworks to a real interface. Working through a heuristic analysis systematically showed how structured evaluation reveals problems that informal observation misses.

**Suggestions for improvement:** Comparing the group's evaluation with feedback from actual users of the interface would have validated — or challenged — the findings and made the analysis more grounded.

**Other thoughts:** This assignment trained me to look at interfaces critically. Now, when I use software that feels awkward or confusing, I can usually identify which usability principle has been violated — and that awareness guides better design decisions in my own work.
