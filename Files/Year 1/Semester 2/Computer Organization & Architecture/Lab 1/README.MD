## 📖 What I Did

I completed the guided lab tasks, working through exercises that introduced the components of a computer system — CPU, memory, input/output — and how they interact during program execution.

---

## 💭 Reflection

**What I gained:** An initial understanding of how hardware components work together. The lab made the lecture content tangible by connecting abstract descriptions to observable behaviour.

**Suggestions for improvement:** Providing simulation software that lets students trace an instruction through all CPU stages would make the learning more interactive and memorable.

**Other thoughts:** Lab 1 set the context for everything that followed in this course. Getting a clear picture of the big-picture structure of a computer early on made the more detailed topics easier to place.
