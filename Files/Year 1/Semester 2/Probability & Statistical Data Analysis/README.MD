# 📘 Probability & Statistical Data Analysis

**Year 1 · Semester 2**

This course built the mathematical foundation for reasoning about data — covering probability theory, descriptive and inferential statistics, distributions, hypothesis testing, and regression analysis.

## 📂 Work

No assignment or lab subfolders recorded for this course.

---

## 📖 What I Learned

Probability & Statistical Data Analysis introduced the tools for making sense of uncertain and variable data. I learned to calculate probabilities, describe datasets using summary statistics, work with common probability distributions (normal, binomial, Poisson), and draw conclusions from data through hypothesis testing and confidence intervals.

Regression analysis gave me a first look at modelling relationships between variables — a concept that reappears throughout data engineering and machine learning work.

---

## 💭 Reflection

**What I gained:** A mathematical vocabulary for describing and reasoning about data. Understanding variance, distributions, and hypothesis testing has been directly useful in every data-related course and project since.

**Suggestions for improvement:** More worked examples connecting statistical concepts to real datasets — rather than purely abstract problems — would have made the material feel more immediately applicable. Seeing statistics applied to data I cared about would have been more motivating.

**Other thoughts:** This was one of the harder courses in Year 1, but also one of the most important in hindsight. Almost everything in data engineering involves some level of statistical thinking — whether evaluating model performance, checking data quality, or summarising distributions. The foundation built here has proven consistently valuable.
