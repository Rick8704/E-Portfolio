## 💭 Reflection

**What I gained:** Building a complete C++ application from scratch — rather than just writing isolated exercises — required me to think about program structure as a whole. Designing classes with clear responsibilities and managing interactions between objects pushed my understanding of OOP from concept to practice.

**Challenges faced:** Managing dynamic memory correctly — ensuring every `new` had a matching `delete` and avoiding dangling pointers — was the most demanding part. It instilled a habit of thinking carefully about object lifetimes and ownership that has been useful ever since.

**Other thoughts:** The project marked the point where I moved from writing code that works to writing code that is structured. Thinking in terms of objects, responsibilities, and interfaces is a mindset shift that this project helped cement, and it made transitioning to Java and Python in later courses significantly easier.
