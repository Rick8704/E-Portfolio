# 🔢 Digital Logic — Project

**Year 1 · Semester 1 · Digital Logic**

A lift (elevator) control system designed and simulated using DEEDS (Digital Electronics Education and Design Suite) logic design software.

---

## 🎯 Overview

This project applied digital logic design principles to simulate a working lift system. The system handles floor requests, direction control, and door logic using combinational and sequential circuits built from logic gates.

---

## 🛠️ Tools and Technologies

- DEEDS logic design software
- Boolean algebra and combinational circuit design
- PDF documentation

## 📖 What I Did

I designed and implemented a lift control circuit using DEEDS, applying the logic design principles covered in lectures. The circuit handles inputs from floor request buttons and outputs signals to control the lift motor direction and door state. I documented the design process and verified the circuit behaviour through simulation.

---

## 💭 Reflection

**What I gained:** Practical experience translating a real-world problem into a digital circuit design. Seeing the circuit simulate correctly in DEEDS was a satisfying moment that made the theoretical content of the course feel meaningful.

**Suggestions for improvement:** More time to explore edge cases and failure conditions in the simulation would have deepened my understanding. Testing boundary conditions is something I now approach more carefully in all projects.

**Other thoughts:** This project showed me that good design requires thinking about all possible states and inputs, not just the happy path. That mindset — considering edge cases and verifying each component — has become a habit in my broader technical work.
