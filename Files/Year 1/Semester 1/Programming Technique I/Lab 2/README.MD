## 💭 Reflection

**What I gained:** A stronger grasp of control flow and how loops change a program's behaviour. Understanding the difference between different loop types and when to use each one became clearer through hands-on practice.

**Suggestions for improvement:** More varied examples using the same concepts would help. Seeing `for` loops applied to different problem types — not just counting — would broaden understanding earlier.

**Other thoughts:** Lab 2 was the point where programming started to feel natural. Completing exercises without needing to look up every syntax detail was a small but meaningful sign of progress.
