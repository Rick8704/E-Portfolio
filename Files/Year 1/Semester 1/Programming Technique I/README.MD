## 📖 What I Learned

Programming Technique I was my first formal introduction to writing code. I learned to structure a program logically, use variables and control flow, write and call functions, and work with arrays. The lab exercises provided structured practice, while the assignments required independent problem-solving.

By the end of the course, I could write C++ programs that solved real input/output problems and understood how to decompose a task into smaller functions.

---

## 💭 Reflection

**What I gained:** A working understanding of structured programming and the C++ language. The habit of writing clean, readable code — using meaningful names, commenting logic, and testing carefully — started here.

**Suggestions for improvement:** More debugging exercises would have been beneficial. Learning to read error messages and trace through code systematically is a skill that takes practice, and dedicated sessions for it would have accelerated my development.

**Other thoughts:** This course sparked genuine interest in programming. The satisfaction of writing a program that produces the correct output motivated me to explore beyond the coursework. That curiosity has driven most of my learning since.
