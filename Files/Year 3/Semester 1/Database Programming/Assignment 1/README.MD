## 💭 Reflection

**What I gained:** Hands-on experience with advanced SQL features that go beyond what most introductory database courses cover. Writing stored procedures and triggers inside the database — rather than handling logic purely in application code — requires a different way of thinking about where logic should live.

**Suggestions for improvement:** Providing a partial implementation to complete — rather than starting from scratch — would allow more focus on the advanced programming concepts rather than schema setup time.

**Other thoughts:** This assignment reinforced that databases are not just storage — they are programmable systems with their own logic, events, and execution environment. That understanding shapes how I design data systems.
