## 💭 Reflection

**What I gained:** Experience building a structured dashboard that serves a business audience — not just a technically correct one. Understanding how to choose the right visualisation type, how to arrange a dashboard for readability, and how to handle interactive filtering took more thought than expected.

**Suggestions for improvement:** Feedback sessions at the Phase 2 milestone — where groups present their dashboard draft and receive structured critique — would improve the quality of Phase 3 significantly. Iterating based on feedback is a core BI skill.

**Other thoughts:** Phase 2 was where the project started to feel like something real. Seeing data that had been carefully prepared and modelled appear as clear, meaningful visualisations in Power BI made the earlier preparation work feel worthwhile.
