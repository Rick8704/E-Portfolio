## 📖 What I Learned

Data Analytics Programming introduced Python as the primary tool for data work — covering data manipulation with pandas, visualisation with matplotlib, and scripting analytical workflows. This was the beginning of my Python proficiency.

Data Mining explored techniques for discovering patterns in large datasets — classification, clustering, association rules, and model evaluation. The course bridged statistics and machine learning, showing how raw data can be transformed into actionable insight.

Object Oriented Programming formalised OOP principles in Java — classes, inheritance, polymorphism, encapsulation, and design patterns. Operating Systems covered process management, memory allocation, scheduling, and file systems — explaining what happens beneath every program.

---

## 💭 Reflection

This semester brought data engineering into sharper focus. Data Analytics Programming and Data Mining together showed what it means to work with data professionally — cleaning it, exploring it, modelling it, and communicating findings.

Python became my preferred language this semester. Its readability and the richness of its data science ecosystem made it a natural fit for the work I wanted to do. The OOP course also improved how I structure larger programs.

Operating Systems was demanding but valuable. Understanding how processes and memory are managed at the OS level made me a more thoughtful programmer and helped me reason about performance and resource usage in data pipelines.
