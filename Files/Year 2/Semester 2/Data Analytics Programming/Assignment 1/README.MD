## 📖 What I Did

I loaded a dataset into a pandas DataFrame, explored its structure and contents, performed cleaning operations to handle missing or inconsistent values, computed summary statistics, and produced visualisations to communicate key findings. The analysis was structured and documented within the Jupyter Notebook.

---

## 💭 Reflection

**What I gained:** Familiarity with the core data analytics workflow in Python — loading, exploring, cleaning, analysing, and visualising a dataset. This assignment established the foundation for all subsequent Python data work.

**Suggestions for improvement:** Providing datasets with intentional quality issues — missing values, inconsistent formats, duplicates — would give students more realistic practice in data cleaning from the start.

**Other thoughts:** Working in a Jupyter Notebook for the first time was a revelation. The ability to mix code, output, and written explanation in a single document made the analysis feel more coherent and communicable than a standalone script.
